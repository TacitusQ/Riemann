# Riemann

Hello World!

This program calculates polynomial terms in an infinite expansion 
which are useful in analytically continuing the Riemann Zeta Function
for negative integers.

Sample output:
Riemann q$ ./series 5

[ 1 ][ 0 ][ 0 ][ 0 ][ 0 ]
[ 1 ][ 1/2 ] [ -1/6 ] [ 1/24 ] [ -1/120 ] 
[ 1 ][ 1/2 ] [ 1/12 ] [ -1/8 ] [ 11/180 ] 
[ 1 ][ 1/2 ] [ 1/12 ] [ 0 ][ -23/360 ] 
[ 1 ][ 1/2 ] [ 1/12 ] [ 0 ][ -1/720 ]
